# my-vibe-page

Personal aggregator page built with Next.js 16 + Tailwind CSS v4 + Framer Motion. Deployed on Cloudflare Pages.

## Stack
- Next.js 16.2.4 (App Router, Turbopack)
- Tailwind CSS v4 (CSS-based config via `@tailwindcss/postcss`)
- Framer Motion 12
- Cloudflare Pages via `@opennextjs/cloudflare`

## Commands
- `npm run dev` — local dev server
- `npm run build` — standard Next.js build
- `npm run cf:build` — build for Cloudflare Pages
- `npm run cf:preview` — preview Cloudflare Pages output locally

## Project structure
- `app/` — Next.js App Router (layout, page, API routes)
- `components/` — Client components (Hero, SocialCards, MusicCard, InterestTags, FootprintMap, PhotoGallery, LiveLocation)
- `public/data/music.json` — music data (generated by GitHub Action)
- `public/photos/` — photo gallery images
- `app/api/location/` — edge runtime, reads CF request headers (cf-ipcountry, cf-region)
- `app/api/photos/` — node runtime, reads filesystem for photo list

## Design
- Dark mode only (#08090a background)
- Glassmorphism cards with backdrop-blur
- Apple-inspired typography (SF Pro / PingFang SC / system sans-serif)
- Framer Motion hover lift effects on all cards
